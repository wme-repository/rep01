# Plan 07-01: Modern UI with Tailwind CSS

**Wave:** 1
**Depends on:** None
**Files modified:** `frontend/package.json`, `frontend/vite.config.js`, `frontend/src/App.jsx`, `frontend/src/index.css`, `frontend/src/components/*.jsx`

<objective>
Upgrade frontend with Tailwind CSS and shadcn/ui components while maintaining all existing functionality.
</objective>

<read_first>
- `frontend/src/App.jsx` — current app structure
- `frontend/src/components/Dashboard.jsx` — existing dashboard
- `frontend/package.json` — current dependencies
</read_first>

<action>
1. **Install Tailwind CSS:**
   ```bash
   cd frontend && npm install -D tailwindcss postcss autoprefixer
   npx tailwindcss init -p
   ```

2. **Configure Tailwind** (`tailwind.config.js`):
   ```js
   export default {
     content: ['./index.html', './src/**/*.{js,jsx}'],
     theme: {
       extend: { colors: { slate: { 850: '#1e293b' } } }
     },
     plugins: [],
   }
   ```

3. **Add Tailwind directives** to `frontend/src/index.css`:
   ```css
   @tailwind base;
   @tailwind components;
   @tailwind utilities;
   ```

4. **Install shadcn/ui dependencies:**
   ```bash
   cd frontend && npm install @radix-ui/react-slot @radix-ui/react-dialog @radix-ui/react-tabs @radix-ui/react-label class-variance-authority clsx tailwind-merge lucide-react
   ```

5. **Refactor App.jsx** with Tailwind classes and tabs navigation

6. **Update all components** (Dashboard, PipelinePanel, ArticleTable, HealthCheck) with Tailwind styling

7. **Update vite.config.js** to handle PostCSS properly
</action>
